Conceitos Fundamentais e HTML

Tecnologias

Nesse início da aula, são apresentadas as tecnologias necessárias para o desenvolvimento de sites ou páginas,são eles:

  • HTML5
  • CSS3
  • JavaScript

O HTML5 é responsável por todo o conteúdo: Texto,Tabela,Imagem e Video. Já o CSS3 está totalmente ligado a estilização do conteúdo: Cores, Fontes, Tamanho, Sombras e Posicionamento. Por fim, o JavaScript tem a função de interatividade, através de Menus interativos, Validação, Pop Ups e Animação.

Visão Geral dos Códigos

Nesse momento, são apresentadas as estruturas básicas de cada ferramenta.

HTLM
É indicado através de tags, sua principal característica é o html no início e fim do código.
CSS
Vai trabalhar em conjunto com o HTML, ele é responsável pela estilização do conteúdo e atua como seletor, ou seja, encontra no HTML o texto necessário para modificação.
JavaScript
Vai trabalhar diretamente com a lógica.

Funcionamento das Tecnologias

Uma breve explicação de como vão funcionar essas tecnologias para entendermos o que ocorre quando visitamos uma página da web. Ao abrir o navegador e digitar a URL, uma requisição é disparada ao servidor DNS que retorna o IP da página em um primeiro momento. Logo após, uma requisição é designada ao servidor do site e ele disponibiliza o HTML,CSS e JavaScript retornando para o cliente. A partir desse processo, o navegador vai interpretar constantemente esse conteúdo.

Pagina da Web e WebSite

A página da Web basicamente é um arquivo onde você pode inserir textos,imagens,tabelas,planilhas,apresentações, entre outros, que permitem a navegação de uma página ou seção.

WebSites ou Sites são uma coleção de páginas armazenadas em uma pasta ou subpasta relacionadas ao servidor web.

Arquitetura Cliente-Servidor e Profissões

Nesse tópico, há uma breve explicação de como funciona a conexão entre as pessoas e os servidores da web. Primeiro, o cliente faz a requisição ao servidor, ocorre o processamento(chamado de SERVER-SIDE) e retorna para o cliente(chamado de CLIENT-SIDE).

O Server Side deve haver tecnologias capazes de atender todas as necessidades dos clientes, assim como o Client Side deve ter o melhor retorno com as tecnologias corretas.

Os profissionais dessas áreas se subdividem em: Front-End, trabalhando em prol do cliente, os Back-End, ao lado dos servidores e também os Full Stack, que são uma junção de ambos.

HTML - Hypertext Markup Language

É uma linguagem de Marcação de Hipertexto baseada em tags. Criada em 1990, por Tim Berners-Lee para publicação de conteúdo em hipertexto. Entre elas estão os textos,imagens, audios e videos. Podem serem vinculadas através de hyperlinks.

Tags

Definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas. Sem o uso delas, o navegador exibe um texto plano. Podem também possuir atributos, ou seja, parâmetros que definem algum tipo de informação extra ao navegador.
