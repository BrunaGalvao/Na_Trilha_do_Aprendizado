HIPERLINKS

Os Hiperlinks ou Links são uma referência dentro de um documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico.

Os links podem ser:

  1. Remotos;
  2. Internos;
  3. Locais;
  4. Com Endereço de Correio ou Arquivos.

Marcação de Hiperlinks

Acontece quando utilizamos a tag < a > e a propriedade href que faz a referência para o conteúdo desejado.

Sintaxe:

< a href = "endereço" > Conteúdo Referenciado Clicável < /a >

Hiperlinks Externos ou Remotos

São links que referenciam sites ou páginas da Web. Na propriedade href utilizamos a URL (Uniform Resource Locator) para referência que desejamos.

Sintaxe:

< a href = "URL" > Conteúdo Referênciado < /a >

A sintaxe pode ser complementada com os seguintes itens:

Hiperlinks Internos

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documento é extenso.

Sintaxe:

< a href = "#referencia" > Conteúdo Referênciado < /a >

< a name = "referencia" > < /a >

Hiperlinks Locais

São links que apontam para outras páginas dentro de um mesmo site.

Sintaxe:

< a href = "arquivo.html" > Conteúdo Referenciado < /a >

Hiperlinks em Imagens

É possível realizar hiperlinks envolvendo a tag de imagem com a tag de hiperlink.

Sintaxe:

< a href = "referencia" > < img src = "imagem.jpg" > < /a >

Hiperlinks para Download de Arquivos

São links que apontam para arquivos e possibilitam realizar o Download destes arquivos.

Sintaxe:

< a href = "arquivo.extensao" download ="arquivo.extensao" type ="tipo" > Conteúdo < /a >

Hiperlinks para Endereço Eletrônico

São links que apontam para um endereço eletrônico

Sintaxe:

< a href="mailto:email@email.com > email@email.com < /a >
