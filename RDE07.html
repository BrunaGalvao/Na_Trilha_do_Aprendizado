Branches e Hospedagem

Branches

São ramificações de um projeto de software ou documento e tem como o objetivo o desenvolvimento pararelo de:

  • Novas Funcionalidades;
  • Correção de Problemas;
  • Alterações de Códigos.

Branch Master ou Main

É o braço principal de um projeto em um repositório.

Branch Secundária

É o braço secundário de um projeto em um repositório.

Hospedagem

A hospedagem é um serviço que possibilita as pessoas ou empresas com sistemas online a guardar páginas,arquivos,informações,imagens,videos ou qualquer outro conteúdo acessível por Web.

O Github Pages oferece hospedagem de forma gratuíta de sites. Com a sua conta e um repositório público, será disponibilizado uma URL com o seguinte padrão: nomeusuario.github.io/repositorio.

Como Realizar Branches no GitHub?

Primeiro Passo

Após a criação da conta no site Github e a sua vinculação com o Github Desktop, acesse o seu login onde estarão disponíveis todos os seus repositórios.

Próximos Passos

Em seu GitHub Desktop crie um repositório, conforme a imagem abaixo:

Publique seu repositório e não esqueça de deixar a licença pública!

Abra o seu repositório externo e crie um arquivo INDEX.HTML

No seu GitHub Desktop, dê um COMMIT para a validação do seu código, isso é indispensável para criar uma MAIN principal, que será vinculada as suas Branches Secundárias.

Após esse processo, vamos realizar a criação das Branches Secundárias. Elas são de extrema importância para a elaboração do código principal MAIN, nelas podemos realizar as alterações que serão vinculadas ao final.

Essa criação é fundamental para a manipulação dos códigos. Após a criação dessa Branch Secundária, será realizado o desenvolvimento do código no repositório externo e a repetição dos processos acima.

Concluindo todos esses processos, precisamos acessar o nosso MAIN e vinculá-lo com a nossa Branch Secundária, essa etapa é realizada por meio da opção Merge into current branch.

Realizado esse processo, agora o código MAIN estará totalmente alterado de acordo com as Branches. Seguindo essas etapas, o seu projeto estará sendo desenvolvido da forma mais correta e segura.

Como Realizar a Hospedagem?

Seguindo o processo citado acima, em sua conta GitHub, localize seu repositório e acesse a aba de Settings.

Em GitHub Pages, selecione a Branch Main e clique em Save. A sua hospedagem será gerada no padrão de URL.

Por fim, o seu projeto desenvolvido estará disponível após a criação dessa hospedagem gratuíta. Qualquer alteração desejada, o processo se repetirá e a atualização acontecerá.
